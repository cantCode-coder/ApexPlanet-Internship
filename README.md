# ApexInternship Projects

A collection of web development projects showcasing HTML, CSS, and JavaScript skills with progressive complexity. All projects are responsive and feature interactive functionality.

---

## Project Overview

### Task1: Explore India - Travel Destinations
**Purpose:** Interactive travel guide featuring Indian destinations with flip card animations

**Key Features:**
- Flip card animation effect for destinations
- Hero section with call-to-action
- Navigation menu with smooth scrolling
- Destination cards with images and descriptions
- Scroll-to-top button functionality
- Random travel suggestions generator

**Technologies:** HTML, CSS (animations & transforms), JavaScript

**Files:**
- index.html - Structure with destination cards
- style.css - Styling and flip card animations
- script.js - Suggestion generator and scroll-to-top logic

---

### Task2: Web Development Fundamentals
**Purpose:** Demonstration of basic web development concepts with practical applications

**Key Features:**
- Contact form with validation (email pattern matching)
- To-Do list application (add/remove tasks)
- Grid layout showcasing HTML, CSS, JavaScript technologies
- Form submission handling with error checking
- DOM manipulation for dynamic task management

**Technologies:** HTML, CSS, JavaScript (DOM manipulation)

**Files:**
- index.html - Form and to-do list structure
- style.css - Layout and styling
- style.js - Form validation and task management

---

### Task3: Advanced Styling & JavaScript
**Purpose:** Advanced interactive features with API integration

**Key Features:**
- JavaScript Quiz with multiple choice questions
- Answer validation with visual feedback (green for correct, red for incorrect)
- Random Joke Generator using external API
- Responsive design with modern styling
- API fetch integration (Official Joke API)
- Quiz auto-progression after answer submission

**Technologies:** HTML, CSS, JavaScript (Fetch API, async/await concepts)

**Files:**
- index.html - Quiz and joke generator structure
- style.css - Responsive design and styling
- style.js - Quiz logic and API fetch functionality

---

### Task4: Watch Inventory Management
**Purpose:** E-commerce inventory system with filtering and sorting capabilities

**Key Features:**
- Watch inventory with 4 brands (Rolex, Omega, Seiko, Apple Watch)
- Brand-based filtering (All/Rolex/Omega/Seiko/Apple)
- Price-based sorting (Low to High, High to Low)
- Sidebar navigation (Home, Inventory, Tasks, Contact)
- Task management with localStorage persistence
- Card-based product display with badges
- Statistics dashboard showing brand and model counts

**Technologies:** HTML, CSS, JavaScript (Array methods, localStorage, DOM manipulation)

**Files:**
- index.html - Layout with sidebar and inventory grid
- style.css - Modern styling for inventory interface
- style.js - Filtering, sorting, task persistence logic

---

### Task5: Personal Portfolio
**Purpose:** Professional portfolio website with smooth navigation and animations

**Key Features:**
- Smooth scroll navigation with active link highlighting
- AOS (Animate On Scroll) library for section animations
- Hero section with CTA buttons (View Work, GitHub)
- About section with professional info and tags
- Skills, Projects, and Contact sections
- Sidebar navigation with icon-based menu
- Responsive design with Font Awesome icons
- Professional layout for showcasing developer skills

**Technologies:** HTML, CSS, JavaScript, AOS library, Font Awesome icons

**Files:**
- index.html - Portfolio sections and structure
- style.css - Modern styling and responsive design
- style.js - Smooth scroll navigation and AOS initialization
